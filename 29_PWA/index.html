Progressive Web Apps (PWA)

Esercitazione sulle Progressive Web Apps in JavaScript

Introduzione

Le Progressive Web Apps (PWA) combinano il meglio delle applicazioni web e delle app native, offrendo un'esperienza utente superiore con funzionalità offline, installabilità e prestazioni migliorate.

In questa esercitazione, esploreremo i concetti fondamentali delle PWA e impareremo a implementarle utilizzando JavaScript.

Esempi Pratici

Esempio 1: PWA Base con Manifest e Service Worker Esempio 2: Strategie di Caching e Installazione

Materiale Teorico

1. Introduzione alle PWA 2. Web App Manifest 3. Service Workers per PWA 4. Strategie di Caching 5. Esperienza di Installazione 6. Lighthouse e Audit

Esercizi

  1. Crea un Web App Manifest per un'applicazione web esistente
  2. Implementa un Service Worker che memorizza nella cache le risorse essenziali
  3. Aggiungi l'esperienza di installazione a una PWA
  4. Crea una PWA completa con funzionalità offline, manifesto e icone
  5. Esegui un audit con Lighthouse e ottimizza il punteggio PWA
