üî¢ \d e \D - Cifre e Non-Cifre

Classi per le cifre:

\d - Matcha qualsiasi cifra [0-9]

\D - Matcha qualsiasi carattere NON cifra [^0-9]

// \d - Cifre "Order 123, Item 456".match(/\d+/g); // ["123", "456"] // \D - Non-cifre "Order 123".match(/\D+/g); // ["Order ", " "] // Esempi pratici const phone = "Tel: 123-456-7890"; phone.match(/\d/g).join(''); // "1234567890" const price = "$99.99"; price.match(/\d+\.?\d*/); // "99.99"

üìù \w e \W - Word Characters

Classi per caratteri "word":

\w - Matcha caratteri word: [a-zA-Z0-9_]

\W - Matcha caratteri NON word: [^a-zA-Z0-9_]

Nota: \w include lettere, cifre e underscore

// \w - Word characters "hello_world 123".match(/\w+/g); // ["hello_world", "123"] // \W - Non-word characters "hello, world!".match(/\W+/g); // [", ", "!"] // Username validation const isValidUsername = (name) => /^\w{3,16}$/.test(name); isValidUsername("user_123"); // true isValidUsername("user@123"); // false // Extract email parts "user@example.com".match(/(\w+)@(\w+)\.(\w+)/); // ["user@example.com", "user", "example", "com"]

‚¨ú \s e \S - Whitespace

Classi per spazi bianchi:

\s - Matcha spazi bianchi: [ \t\n\r\f\v]

\S - Matcha caratteri NON spazio

Include: spazio, tab, newline, carriage return

// \s - Whitespace "hello world".match(/\s+/g); // [" "] // \S - Non-whitespace "hello world".match(/\S+/g); // ["hello", "world"] // Normalizzare spazi multipli const text = "hello world test"; text.replace(/\s+/g, ' '); // "hello world test" // Contare parole const countWords = (str) => str.match(/\S+/g).length; countWords("hello world test"); // 3

üìä [a-z] - Intervalli di Caratteri

Definire range personalizzati:

[a-z] - Lettere minuscole

[A-Z] - Lettere maiuscole

[0-9] - Cifre (equivale a \d)

[a-zA-Z] - Tutte le lettere

[a-zA-Z0-9] - Lettere e cifre

// Range di caratteri "Hello World 123".match(/[a-z]+/g); // ["ello", "orld"] "Hello World 123".match(/[A-Z]+/g); // ["H", "W"] "Hello World 123".match(/[A-Z][a-z]+/g); // ["Hello", "World"] // Range esadecimale "Color: #FF5733".match(/#[0-9a-fA-F]{6}/); // ["#FF5733"] // Combinazioni "user123@test.com".match(/[a-z0-9]+/g); // ["user123", "test", "com"]

üö´ [^...] - Negazione e Combinazioni

Negare set di caratteri:

[^abc] - Qualsiasi carattere TRANNE a, b, c

[^0-9] - Qualsiasi carattere NON cifra (equivale a \D)

[^a-zA-Z] - Qualsiasi carattere NON lettera

// Negazione "hello123".match(/[^0-9]+/g); // ["hello"] "hello123".match(/[^a-z]+/g); // ["123"] // Estrarre solo vocali "hello world".match(/[aeiou]/gi); // ["e", "o", "o"] // Estrarre solo consonanti "hello".match(/[^aeiou\s\d\W]/gi); // ["h", "l", "l"] // Rimuovere caratteri speciali HTML "
