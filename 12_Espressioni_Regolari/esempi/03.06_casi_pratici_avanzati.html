üìë Parser CSV/TSV con Validazione

Parsing dati tabulari:

Gestisce virgolette, escape, tipi di dato

// CSV con gestione virgolette e escape const csvPattern = /(?:^|,)(?:"(?<quoted>[^"]*(?:""[^"]*)*)"|(?<unquoted>[^,]*))/g; // Parse con tipi function parseCSVLine(line) { const fields = []; let match; while ((match = csvPattern.exec(line)) !== null) { const value = match.groups.quoted || match.groups.unquoted; fields.push(value.replace(/""/g, '"')); } return fields; }
