üí° Cosa sono i Gruppi di Cattura

Concetto fondamentale:

I gruppi di cattura permettono di:

  • Isolare parti specifiche di un pattern
  • Estrarre sottostringhe dal match
  • Applicare quantificatori a pi√π caratteri
  • Usare i contenuti catturati in replace

Sintassi: (pattern)

// Senza gruppi - matcha solo const text = "335-1234567"; const match1 = text.match(/\d{3}-\d{7}/); console.log(match1); // ["335-1234567"] // Con gruppi - cattura le parti const match2 = text.match(/(\d{3})-(\d{7})/); console.log(match2); // ["335-1234567", "335", "1234567"] // [0] = match completo // [1] = primo gruppo (prefisso) // [2] = secondo gruppo (numero)

üéØ Accesso ai Gruppi Catturati

Metodi di accesso:

match[0] - Match completo

match[1], match[2], ... - Gruppi catturati

exec() - Metodo dell'oggetto RegExp

// Estrazione data DD/MM/YYYY const text = "Data: 25/12/2023"; const pattern = /(\d{2})\/(\d{2})\/(\d{4})/; const match = text.match(pattern); if (match) { console.log(`Giorno: ${match[1]}`); // "25" console.log(`Mese: ${match[2]}`); // "12" console.log(`Anno: ${match[3]}`); // "2023" } // Usando exec() const regex = /(\w+)@(\w+)\.(\w+)/; const result = regex.exec("user@example.com"); console.log(result[1]); // "user" console.log(result[2]); // "example" console.log(result[3]); // "com"

üî¢ Lavorare con Gruppi Multipli

Gruppi annidati e multipli:

Puoi avere molti gruppi nello stesso pattern

I gruppi vengono numerati dall'esterno verso l'interno

Massimo teorico: dipende dall'implementazione (solitamente 99+)

// Gruppi multipli - URL completo const url = "https://www.example.com:8080/path"; const pattern = /(\w+):\/\/([\w.]+):(\d+)(\/\w+)?/; const match = url.match(pattern); console.log(match[1]); // "https" (protocollo) console.log(match[2]); // "www.example.com" (dominio) console.log(match[3]); // "8080" (porta) console.log(match[4]); // "/path" (percorso) // Carta di credito con gruppi const card = "1234-5678-9012-3456"; const cardPattern = /(\d{4})-(\d{4})-(\d{4})-(\d{4})/; const parts = card.match(cardPattern); // parts = ["1234-5678-9012-3456", "1234", "5678", "9012", "3456"]

üîÅ Gruppi con Quantificatori

Applicare quantificatori ai gruppi:

(abc)+ - Ripete l'intero gruppo

(abc){2,4} - Ripete gruppo da 2 a 4 volte

(abc)? - Gruppo opzionale

// Gruppo ripetuto const pattern1 = /(ha)+/; console.log("hahaha".match(pattern1)); // ["hahaha", "ha"] - cattura solo l'ultima ripetizione // Gruppo opzionale const pattern2 = /https?:\/\/([\w.]+)/; console.log("http://example.com".match(pattern2)[1]); // "example.com" console.log("https://example.com".match(pattern2)[1]); // "example.com" // Ripetizione con range const pattern3 = /(\d{2,4})-/g; "12-345-6789".match(pattern3); // ["12-", "345-", "6789-"]
