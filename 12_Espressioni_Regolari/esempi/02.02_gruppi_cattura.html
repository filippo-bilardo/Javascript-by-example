üí° Cosa sono i Gruppi di Cattura

Gruppi con parentesi ():

Le parentesi tonde () creano gruppi di cattura che:

  • Raggruppano parti di un pattern
  • Catturano il testo matchato per riutilizzarlo
  • Permettono di applicare quantificatori a gruppi
  • Possono essere referenziati con \1, \2, \3, etc.
// Senza gruppi "hello world".match(/hello/); // ["hello"] // Con gruppi di cattura const match = "John Smith".match(/(\w+) (\w+)/); // match[0] = "John Smith" (match completo) // match[1] = "John" (primo gruppo) // match[2] = "Smith" (secondo gruppo) // Accesso ai gruppi console.log(`Nome: ${match[1]}, Cognome: ${match[2]}`);

üîÑ Backreferences - Riferimenti ai Gruppi

Riferimenti indietro nel pattern:

\1, \2, \3... - Riferiscono il contenuto catturato dal gruppo 1, 2, 3...

Uso: Trovare ripetizioni, palindromi, tag HTML matchati

// Trovare parole ripetute const pattern = /\b(\w+)\s+\1\b/; pattern.test("hello hello"); // true pattern.test("hello world"); // false // Tag HTML matchati const htmlPattern = /<(\w+)>.*?<\/\1>/; htmlPattern.test("<div>test</div>"); // true htmlPattern.test("<div>test</span>"); // false // Quote matchate const quotePattern = /(['"]).*?\1/; quotePattern.test("'hello'"); // true quotePattern.test("'hello\""); // false

üè∑Ô∏è Gruppi Nominati

Gruppi con nome - ES2018+:

(?<nome>...) - Crea un gruppo con un nome

Accesso: match.groups.nome

Backreference: \k<nome>

// Gruppi nominati const pattern = /(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/; const match = "2023-12-25".match(pattern); // Accesso tradizionale console.log(match[1]); // "2023" console.log(match[2]); // "12" // Accesso per nome (pi√π leggibile) console.log(match.groups.year); // "2023" console.log(match.groups.month); // "12" console.log(match.groups.day); // "25" // Backreference nominato const palindrome = /(?<char>\w)\w*\k<char>/; palindrome.test("radar"); // true

üö´ Gruppi Non-Catturanti

Gruppi che non catturano - (?:...):

(?:...) - Raggruppa senza catturare

Vantaggi: Performance migliori, numeri di gruppo pi√π chiari

Uso: Quando serve raggruppare ma non serve il contenuto

// Con gruppi catturanti const pattern1 = /(\d{3})-(\d{3})-(\d{4})/; const match1 = "123-456-7890".match(pattern1); // match1 = ["123-456-7890", "123", "456", "7890"] // Con gruppi non-catturanti per separatori const pattern2 = /(\d{3})(?:-)(\d{3})(?:-)(\d{4})/; const match2 = "123-456-7890".match(pattern2); // match2 = ["123-456-7890", "123", "456", "7890"] // I separatori "-" non sono catturati // Utile con alternanza const pattern3 = /(?:Mr|Mrs|Ms)\. (\w+)/; const match3 = "Mr. Smith".match(pattern3); // match3 = ["Mr. Smith", "Smith"] // "Mr" non √® catturato

üîÑ Replace con Gruppi

Sostituzioni usando gruppi:

$1, $2, $3... - Riferiscono i gruppi catturati nella stringa di replace

$& - L'intero match

$` - Testo prima del match

$' - Testo dopo il match

// Invertire nome e cognome const name = "John Smith"; const reversed = name.replace(/(\w+) (\w+)/, "$2, $1"); // "Smith, John" // Formattare data const date = "2023-12-25"; const formatted = date.replace(/(\d{4})-(\d{2})-(\d{2})/, "$3/$2/$1"); // "25/12/2023" // Con funzione callback const text = "Price: $100"; const converted = text.replace(/\$(\d+)/, (match, num) => { return `‚Ç¨${num * 0.85}`; }); // "Price: ‚Ç¨85"
