# Esercitazioni Guidate - Oggetti in JavaScript

Questa cartella contiene tre esercitazioni guidate di difficoltà crescente per consolidare la comprensione degli oggetti in JavaScript.

## 📚 Struttura delle Esercitazioni

### 01 - Esercitazione Base
**Livello:** Principiante  
**Tempo stimato:** 30-40 minuti  
**Argomento:** Gestione di una Biblioteca di Libri

**Cosa imparerai:**
- Creare oggetti con proprietà e metodi
- Accedere e modificare proprietà
- Utilizzare metodi per operazioni base
- Lavorare con array di oggetti
- Filtrare e manipolare dati

**Prerequisiti:**
- Conoscenza base di variabili e funzioni
- Concetti base degli array

---

### 02 - Esercitazione Intermedia
**Livello:** Intermedio  
**Tempo stimato:** 45-60 minuti  
**Argomento:** Sistema di Gestione Studenti e Corsi

**Cosa imparerai:**
- Utilizzare costruttori e prototipi
- Implementare ereditarietà
- Destrutturazione e spread operator
- Lavorare con oggetti nidificati
- Applicare metodi avanzati degli array (map, filter, reduce)
- Usare classi ES6

**Prerequisiti:**
- Aver completato l'esercitazione base
- Familiarità con le funzioni e i concetti base OOP

---

### 03 - Esercitazione Avanzata
**Livello:** Avanzato  
**Tempo stimato:** 60-90 minuti  
**Argomento:** Sistema E-Commerce con Pattern di Progettazione

**Cosa imparerai:**
- Applicare pattern di progettazione (Singleton, Factory, Observer)
- Utilizzare getters, setters e proprietà private
- Implementare validazione robusta e gestione errori
- Lavorare con JSON e serializzazione
- Creare un'applicazione completa e modulare
- Best practices per codice professionale

**Prerequisiti:**
- Aver completato le esercitazioni precedenti
- Comprensione solida di OOP e prototipi
- Conoscenza dei concetti avanzati di JavaScript

---

## 🎯 Come Usare le Esercitazioni

### 1. Approccio Consigliato
- **Leggi attentamente** ogni esercizio prima di iniziare
- **Prova a risolvere** senza guardare le soluzioni
- **Usa i test** decommentando le righe indicate per verificare il tuo codice
- **Consulta le guide teoriche** se hai dubbi su concetti specifici
- **Rivedi le soluzioni** solo dopo aver tentato l'esercizio

### 2. Esecuzione
Puoi eseguire le esercitazioni in due modi:

**Nel browser:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>Esercitazione</title>
</head>
<body>
    <h1>Apri la Console (F12)</h1>
    <script src="01_esercitazione_base.js"></script>
</body>
</html>
```

**Con Node.js:**
```bash
node 01_esercitazione_base.js
```

### 3. Struttura di Ogni Esercitazione
Ogni file è organizzato in:
- **Parti tematiche** - Sezioni che raggruppano esercizi correlati
- **Esercizi numerati** - Con obiettivo chiaro e istruzioni dettagliate
- **Commenti TODO** - Indicano dove scrivere il codice
- **Test commentati** - Da decommentare per verificare le soluzioni
- **Soluzioni** - Alla fine del file (nascoste da righe vuote)

## 💡 Suggerimenti per lo Studio

### Per Principianti (Esercitazione 1)
1. Non avere fretta - prenditi il tempo necessario
2. Esegui i test dopo ogni esercizio
3. Sperimenta modificando i valori e osserva i risultati
4. Se ti blocchi, rileggi la guida teorica corrispondente

### Per Studenti Intermedi (Esercitazione 2)
1. Concentrati sulla comprensione dei concetti, non solo sulla sintassi
2. Prova a scrivere il codice senza copiare gli esempi
3. Fai esperimenti: cosa succede se modifico questo parametro?
4. Disegna diagrammi per visualizzare l'ereditarietà

### Per Studenti Avanzati (Esercitazione 3)
1. Pensa all'architettura prima di scrivere codice
2. Considera casi limite e gestione errori
3. Rifletti su come migliorare il codice (performance, leggibilità)
4. Prova a estendere gli esercizi con funzionalità aggiuntive
