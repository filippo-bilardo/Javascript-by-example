3. Event Delegation: Usa focusin/focusout!

Per la delegazione, usa sempre focusin/focusout invece di focus/blur.

Form di Registrazione

// ‚ùå SBAGLIATO - focus non propaga! form.addEventListener('focus', function(event) { // Non verr√† mai chiamato per gli input figli }); // ‚úÖ CORRETTO - focusin propaga! form.addEventListener('focusin', function(event) { const input = event.target; console.log('Focus su:', input.name); input.classList.add('focused'); }); form.addEventListener('focusout', function(event) { event.target.classList.remove('focused'); });
