6. Pattern Avanzato: Delegation Router

Sistema di routing degli eventi basato su data attributes.

// Sistema di routing centralizzato const routes = { 'user/create': (params) => createUser(params), 'user/delete': (params) => deleteUser(params), 'product/update': (params) => updateProduct(params), 'order/cancel': (params) => cancelOrder(params) }; container.addEventListener('click', function(event) { const route = event.target.dataset.route; if (!route) return; const params = JSON.parse(event.target.dataset.params || '{}'); const handler = routes[route]; if (handler) { handler(params); } });
