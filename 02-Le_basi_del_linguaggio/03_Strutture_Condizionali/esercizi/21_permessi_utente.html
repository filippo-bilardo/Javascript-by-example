🛡️ Permessi Utente per Ruolo

Seleziona un utente e osserva quali azioni sono disponibili in base al suo ruolo. La logica dei permessi usa esclusivamente && e ||.

Logica dei permessi (JavaScript)

// Ogni permesso è calcolato combinando ruolo e stato dell'account con && e ||
const attivo = utente !== null && utente.stato !== "banned";
const loggato = utente !== null;
const ruolo = utente?.ruolo;

// Permessi base
const puoLeggere = attivo && loggato;
const puoScrivere = attivo && (ruolo === "admin" || ruolo === "editor");
const puoModificare = attivo && (ruolo === "admin" || ruolo === "editor");
const puoEliminare = attivo && ruolo === "admin";
const puoPubblicare = attivo && ruolo === "admin";
const puoGestireUtenti = attivo && ruolo === "admin";
