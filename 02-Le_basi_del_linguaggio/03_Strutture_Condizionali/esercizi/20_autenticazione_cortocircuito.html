🔐 Autenticazione con Cortocircuito

Un semplice sistema di login che usa && (AND) per concatenare le verifiche e || (OR) per selezionare il messaggio di errore appropriato.

Accedi al sistema

Utenti validi: admin, mario, guest

Password: admin→admin123, mario→pass42, guest→guest

🔍 Passi di valutazione (cortocircuito &&)

Codice corrispondente:
const ok = campiCompilati && utenteEsiste && passwordCorretta;
const errore = (!campiCompilati && "Compila tutti i campi") || (!utenteEsiste && "Utente non trovato") || (!passwordCorretta && "Password errata");
