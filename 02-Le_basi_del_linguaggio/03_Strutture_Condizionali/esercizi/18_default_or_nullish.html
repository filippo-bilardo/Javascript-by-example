⚡ || vs ?? — Valori di Default

Esplora la differenza fondamentale tra l'operatore OR || e l'operatore di coalescenza nulla ?? quando si assegnano valori di default.

Regola chiave:
|| usa il default se il valore è qualsiasi valore falsy (0, "", false, null, undefined, NaN)
?? usa il default solo se il valore è null o undefined
